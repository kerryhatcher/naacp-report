# docs/research

Notes, source lists, and non-PII data discovered while researching for this report — e.g. links to county board pages, meeting schedule notes, official turnout reports, news clippings about board controversies, or aggregated/summary data pulled from public sources.

**Nothing with individual-level PII goes here.** If a source turns out to contain PII (e.g. a raw voter file export), it belongs in [`sensitive-data/`](../../sensitive-data/README.md) instead, which is git-ignored. When in doubt, keep it out of this folder.

This folder is tracked normally in git — everything here is expected to be shared and committed.
